# 虎哥PDF重命名工具

一个简单易用的 PDF 文件批量重命名工具，可以自动提取 PDF 首页最大字体的文本作为文件名。

## 功能特点

- 自动提取 PDF 首页最大字体文本作为文件名
- 支持批量处理多个 PDF 文件
- 智能文本提取和清理
- 自动处理文件名冲突
- 完整的日志记录

## 使用方法

### 安装

1. 下载最新版本的程序
2. 解压到任意目录

### 使用方式

双击运行 `虎哥PDF重命名工具.exe`，在弹出的文件选择窗口中选择要处理的 PDF 文件。

## 注意事项

- 程序会自动在程序目录下创建 logs 文件夹存放日志文件
- 处理过程中按 ESC 键可以终止操作
- 重命名时会自动处理文件名冲突，为重复的文件名添加序号
- 建议在重要文件上使用前先进行备份

## 系统要求

- Windows 7/8/10/11
